## Developing

**Requirements:**

-   Node.js (see version in `.tool-versions`)
-   pnpm

**Useful commands:**

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Lint
pnpm run lint

# Build
pnpm run build

# Preview last build
pnpm run preview
```
